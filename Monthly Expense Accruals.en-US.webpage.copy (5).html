User Guide & Field Definitions

Report Views:

  • Summary Accruals: Full report with all columns including DCL #, CI Number, Status, and all details
  • Expense Accruals Report: Simplified view with 18 key columns focused on costs and charges
  • Switch between views using the tabs above the statistics cards

How to Use:

  • Filtering: Use the dropdown filters to narrow down records by Status, Executive, Month, etc.
  • Search: Use global search to find records by DCL#, CI Number, Customer PO, Customer Name, etc.
  • Sorting: Click any column header to sort (click again to reverse)
  • Export: Click "Download Accruals (AED)" to export - all currency values are automatically converted to AED

Currency Conversion (Export):

  • USD → AED: Rate 3.6725
  • SAR → AED: Rate 0.9793
  • EUR → AED: Rate 4.02
  • GBP → AED: Rate 4.65
  • All charge columns (COO, MOFA, Insurance, Freight, etc.) are converted in the exported file

Field Definitions:

  • Green Cells: Calculated/Formula fields (auto-computed)
  • Red Cells: Missing data (N/A values)
  • Blue Cells: Editable fields (manual entry required)
  • S = System Generated: Automatically populated from Oracle/Dataverse
  • M = Manual Entry: Requires user input
  • Formula: Calculated based on other fields
