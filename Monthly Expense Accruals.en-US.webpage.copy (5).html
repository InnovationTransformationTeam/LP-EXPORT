User Guide & Field Definitions

How to Use This Report:

  • Filtering: Use the dropdown filters to narrow down records by specific criteria
  • Search: Use the global search to find records by DCL#, Customer PO, Customer Name, etc.
  • Sorting: Click on any column header to sort by that column (click again to reverse order)
  • Export: Click "Export to Excel" to download the current filtered data
  • Pagination: Navigate through pages using the controls at the bottom of the table

Field Definitions:

  • Green Cells: Calculated/Formula fields (auto-computed)
  • Red Cells: Missing data (N/A values)
  • Blue Cells: Editable fields (manual entry required)
  • S = System Generated: Automatically populated from Oracle/Dataverse
  • M = Manual Entry: Requires user input
  • Formula: Calculated based on other fields

Important Notes:

  • Unit Actual Freight must be entered manually for accurate cost calculations
  • Currency conversions to AED use a rate of 3.675 (as per stakeholder requirement)
  • Shipment Month is derived from Shipped Orders, not DCL Master sailing date
  • Data refreshes every 5 minutes automatically
